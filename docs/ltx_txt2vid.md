# LTX 文生视频 (`ltx_txt2vid.json`)

> 一段文字 → 2 秒短视频。17 节点，无需 FLUX 阶段，显存友好（16-20GB）。

## 怎么用

1. 拖入 `ltx_txt2vid.json`
2. 节点 13：写 Prompt（同时描述画面+运动）
3. 节点 21：分辨率 + 帧数（默认 768×512, 49 帧）
4. 点击 Queue Prompt

## 关键参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| width/height | 768/512 | 比 I2V 小，确保显存够 |
| 帧数 | 49 | ≈2 秒 @ 25fps |
| steps | 20 | 比 I2V 少 10 步，速度更快 |
| sampler | euler | LTX 标配 |
| NAG scale | 11 | 通用值 |

## 与 FLUX→LTX 对比

| | LTX T2V | FLUX→LTX v2 |
|------|---------|-------------|
| 节点 | 17 | 33 |
| 显存 | 16-20 GB | 22-24 GB |
| 首帧质量 | AI 生成 | **FLUX 生成**（更好） |
| 适合 | 快速预览 | **最终成品** |

## 模型依赖

- LTX 全套：UNet + VAE + Gemma + 投影层 + LoRA
- 不需要 FLUX 系列模型
